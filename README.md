FAARUUN MAALIDHA?

Faaruun qulqulluu Abbaan keenya, uumaan keenya, lafaa fi samii irratti kan ittiin galatteeffamu, kan barumsa irraa argannuu, fi kan ittiin kadhannu ta’ee, bifaa sagalee mi’aawwaa ykn Bifa Yeedalootiin kan darbuudha.

Waaqayyo galateeffadhaa, faaruun kan eebbifamedha waan ta’eef, uumaa keessaniif galata galchuun kan hundaa caaludha jedhamee wangeela irratti illee barreeffamee jira. (Faarsaa Daawwit 146_1, Isaayyaas 6_1-5,) yoo wangeela irraa deemnee dubbisne hiikka faaruu argachuu dandenya.
Faarf 146:1, Ba’uu 15:1-22, Isaay 61:5, Mul’a 5:6-14, Faar 65:1-5

⚫	Faarfannaan kennaa galateeffannaa waaqayyoof dhiyaatuudha

⚫	Siif kennaa galateeffannaa nan dhiyeessa Faar 115:8, Ibr 13:15

⚫	Jechi Faaruu jedhu kan dhufe jecha afaan Gi’iizii irraa gara oromiffaatti fudhatame. 

Afaan Gi’iiziin mazmur kan jedhamu yoo ta’u hiikni isaa ammoo Faarfate, galateeffate kan jedhu yoo ta’u, hiikni isaa inni kallattii ifaan jiru galatoo, yokaan galateeffachuu, gammachiisuu, itti gammaduun faarsuu yokaan yeedalessuu (zeemessuu) jechuudha. Kan biraa ammoo waaqayyoo dachaa dachaadhan galateeffachuu jechuudha.

Koottaa waaqayyoo keenyati haa gammannuu, uumaa keenyaaf, Fayyisaa keenyaafa haa ililchinuu, Uumaan keenya waaqayyoo guddaadha waan ta’eef. Faarsaa Daawwit ( 94:1-3, 116:1-2, 149:-4, 150:1-7, 80:1-3,) Isaayyaas 35:1-6, Maatiyoos 26:30, maarqoos 14:26, qolayasisyaas 3:16, mul’ata Yoohannis 14:1-5, Maatiyoos 21:9, maarqoos 11:9, Luq 19:38) isaan kun ragaa kitaaba qulqulluu irratti, faaruun maal akka ta’e nuuf ibsaniidha.

Akuma gubbaa irratti ibsametti faaruu jechuun weeddessuu jechudha. Weeddessuu jechuun ammoo hiikka kallattiin yoo hiikkamuu sagalee qaama keenya keessa bahu sirritii iyyuun kan dhageessifamu jechudha.

Haa ta’u malee faaruun kan galatoo qofaa osoo hin taanee kan gadda keenya illee ittiin ibsanu, kan ittiin gammachuu keenya ibsannu, kan ittiin Abbootii qulqulloota darban yaadannu dha. (Isaayyaas 6:1-5, Maatiyoos 26:30) yoo dubbisne ni arganna.

Akkasuma faaruun kan haadha qulqulluu, Abbaa Keenya Fayyisaa Addunyaa nuuf deesse, haadha haadholii hundaa ol taate, ergamtoota hundaa ol taate Abbootii hundaa ol taate, Uummatoota hundaa ol taate, uumaa gad kan taate Durbee Maariyaam kan itti galateffanudha.

Akkasuma illee faaruun kan waaqayyoo qofa ittin galataffannu osoo hin taane kan Abbootii daandii waaqayyoo irraa hordofaa turan kan ittiin weedisnu, kan ittii kadhannu, kan ittiin gammachuu isa kadhanne irraa arganne tokko itiin ibsannuu fi waan yaadnee isa kadhanne tokko yeroo nuuf guute Sana galatoo keenya kan ittiin ibsanuudha.
